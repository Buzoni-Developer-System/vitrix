# UI UX RULES

## Filosofia Visual
O VitrinCard deve transmitir sensação de:
- modernidade
- premium
- profissionalismo
- tecnologia
- simplicidade

---

## Mobile First
Toda interface deve ser desenhada primeiro para celular.

Prioridades:
- navegação simples
- botões grandes
- espaçamento confortável
- leitura fácil

---

## Estilo Visual
Preferência por:
- visual clean
- sombras suaves
- bordas arredondadas
- boa hierarquia visual
- espaçamento generoso

Evitar:
- visual poluído
- excesso de cores
- excesso de informação

---

## UX
Toda ação importante deve exigir:
- poucos cliques
- mínimo esforço mental
- fluxo intuitivo

---

## Performance Visual
O sistema deve parecer:
- rápido
- fluido
- leve

---

## Branding
A experiência deve passar sensação de:
"Meu negócio parece mais profissional usando isso."

---

## Referência de Produto
Inspirar-se em:
- produtos SaaS modernos
- landing pages premium
- apps mobile bem polidos

---

## Regra Final
Sempre priorizar:
- experiência premium
- clareza
- simplicidade
sobre complexidade ou excesso de features.
