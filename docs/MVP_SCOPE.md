# MVP SCOPE

## Objetivo do MVP
Construir a primeira versão validável do VitrinCard para apresentação e uso real.

O foco é:
- validar mercado
- validar utilidade
- gerar valor percebido imediato

Não é objetivo:
- construir versão final
- construir SaaS completo
- construir sistema enterprise

---

## Funcionalidades Obrigatórias MVP

### Página Pública
- Logo/foto/banner
- Nome comercial
- Descrição/slogan
- Botões rápidos de contato
- Lista de produtos
- Exibição de preço
- Botão WhatsApp

---

### Admin
- Login
- CRUD de produtos
- CRUD de links de ação
- Edição de branding visual
- Edição de informações comerciais

---

### Comercial
- Preço manual
- Sugestão de cotação do dólar
- Dólar manual editável
- Margem global opcional

---

## Funcionalidades Fora do MVP

### Não implementar:
- Multi usuários
- Assinaturas
- Dashboard analítico
- Estoque
- ERP
- Financeiro complexo
- Importação em lote
- Scraping de fornecedor
- Automações agendadas
- Cupons/descontos
- Gestão avançada de pedidos

---

## Filosofia MVP
Se uma funcionalidade:
- não agrega valor imediato
- não impacta validação
- ou aumenta muito complexidade

Ela NÃO deve entrar no MVP.
